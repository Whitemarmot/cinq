# Notification Sounds

This folder contains notification sounds for Cinq.

## Required files:

1. **message.mp3** - Sound for new messages (soft, subtle)
2. **ping.mp3** - Sound for pings (slightly more noticeable)

## Specifications:

- Format: MP3 (for best compatibility)
- Duration: < 2 seconds
- Size: < 50KB per file
- Volume: Normalized, not too loud

## Recommended sources (royalty-free):

- https://notificationsounds.com/
- https://mixkit.co/free-sound-effects/notification/
- https://www.zapsplat.com/

## Tips:

- Keep sounds short and non-intrusive
- Test on both mobile and desktop
- Consider accessibility (don't rely solely on sound)
