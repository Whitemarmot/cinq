# Keyboard Shortcuts — Cinq

Raccourcis clavier globaux disponibles dans l'application Cinq.

## Navigation rapide

| Raccourci | Action |
|-----------|--------|
| `Ctrl + 1` | Ouvrir le chat avec le 1er contact |
| `Ctrl + 2` | Ouvrir le chat avec le 2e contact |
| `Ctrl + 3` | Ouvrir le chat avec le 3e contact |
| `Ctrl + 4` | Ouvrir le chat avec le 4e contact |
| `Ctrl + 5` | Ouvrir le chat avec le 5e contact |

> **Note :** Les numéros correspondent à l'ordre d'affichage dans la grille "Tes 5".

## Actions principales

| Raccourci | Action |
|-----------|--------|
| `Ctrl + N` | Nouveau message (ouvre l'onglet Tes 5 pour choisir un contact) |
| `Ctrl + F` | Ouvrir la recherche |
| `Ctrl + K` | Ouvrir/fermer la recherche (alternatif) |

## Quick Share (partage rapide d'images)

| Raccourci | Action |
|-----------|--------|
| `Ctrl + V` | Coller une image depuis le presse-papier (dans le composer ou le chat) |

> **Note :** Fonctionne avec les captures d'écran, images copiées, etc. Formats supportés : JPEG, PNG, GIF, WebP. Taille max : 5 Mo.

## Fermeture des modals

| Raccourci | Action |
|-----------|--------|
| `Escape` | Fermer le modal/overlay actif |

L'ordre de priorité pour Escape :
1. Image en plein écran
2. Overlay de recherche
3. Panel de chat
4. Modal cadeau
5. Modal ajout contact
6. Modal suppression de compte
7. Menus déroulants des posts

## Accessibilité

- Tous les raccourcis utilisent `Ctrl` (ou `Cmd` sur Mac)
- Les raccourcis ne s'activent pas si un champ de texte est focus (sauf Escape)
- Navigation au clavier complète avec Tab/Shift+Tab
- Skip link disponible pour aller directement au contenu
